Ham Radio

I'm a CEPT-1/Amateur Radio Extra ham radio operator since 1996. My calls are SM6WJM, AD8KM and OZ3ALB.

I'm interested in SDR (Software Defined Radio) and am a contributor to the GNURadio project. Most of my recent work have gone into optimizing GNURadio for ARM CPU based SoC boards, in particular Raspberry Pi 3 and 4. You can check the progress of that work in this issue.

Software

I've been enjoying programming since QBasic 4.5, released in 1988 which also was the last version of QuickBasic. While a horrible language it certainly had some merits when learning programming. In particular you didn't have to set up an advanced development environment. Since most of my code nowadays is related to signal processing it's mostly in C and C++ but but I'm also interested in functional programming. Erlang is a very interesting language.

Currently I'm a Senior software/embedded engineer at Sensrad AB.
