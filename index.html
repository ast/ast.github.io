Who am I?

I'm a Swedish software engineer, anesthesiologist, father and tinkerer. I'm interested in signal processing, ham radio, software and bicycles.

Open source work

  • SoapyAirspyHF - SoapySDR module for the Airspy HF+ Discovery
  • SoapyAirspy - SoapySDR module for the Airspy R2 and Mini
  • Volk - Vector-Optimized Library of Kernels. I contributed a lot of the ARM NEON code.

Ham Radio

I've been a CEPT-1/Amateur Radio Extra operator since 1996, with callsigns SM6WJM, AD8KM, and OZ3ALB. I'm passionate about SDR (Software Defined Radio) and actively contribute to the GNURadio project, focusing on optimizing it for ARM-based boards like the Raspberry Pi 3 and 4.

Software

I’ve been programming since QBasic 4.5, and while it had its quirks, it was a great introduction to coding without needing a complex setup. Today, my work centers around signal processing, primarily using C and C++, but I also explore functional programming, particularly in Erlang.
